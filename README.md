# Regional Traffic Explorer

The **Regional Traffic Explorer** is a public-facing dashboard prototype for exploring traffic activity across the Omaha-Council Bluffs metropolitan area.

The tool brings together regional traffic patterns, top intersections, major interchanges, and multimodal transportation findings in a more visual and accessible format.

## About the Dashboard

This is designed to make regional traffic information easier to explore through:


- Ranked intersection and interchange views
- Regional traffic pattern highlights
- Mobile-friendly layouts

## Data Sources

The dashboard is based on MAPA’s 2021–2022 regional traffic reports, including:

- Regional Traffic Patterns
- Top Traffic Intersections
- Top Traffic Interchanges

Some map and chart elements are placeholders and will be updated as additional GIS layers or source tables become available.
